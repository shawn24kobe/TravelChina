# TravelChina

A lightweight, responsive HTML5 travel showcase site for GitHub Pages. This repository contains an English-language landing page and nine city detail pages for international visitors.

## Features

- Responsive layout optimized for mobile, tablet, and desktop
- City showcase pages for Beijing, Shanghai, Chongqing, Guilin, Guangzhou, Chengdu, Zhangjiajie, Hangzhou, and Xi'an
- Disqus comment section placeholder on each city detail page
- Clean, modern design inspired by travel portals without booking or service clutter

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. In repository settings, enable GitHub Pages from the `main` branch and `/ (root)` folder.
3. Visit the published site URL shown in GitHub Pages settings.

## Customize Disqus

Open any city page and replace `YOUR_DISQUS_SHORTNAME` with your Disqus shortname in the script block under `div id="disqus_thread"`.
